# Background Images for Virtual Backgrounds

This directory contains background images for virtual backgrounds in the video meeting app.

To use your own backgrounds:
1. Add JPG or PNG images to this directory
2. Name them descriptively (e.g., "office.jpg", "beach.jpg", "mountain.jpg")
3. The recommended resolution is 1920x1080 pixels

Sample backgrounds should include:
- office.jpg - An office environment with a desk and window view
- beach.jpg - A beach scene with ocean and sand
- mountain.jpg - A mountain landscape

Please make sure you have the right to use any images you add to this directory.
